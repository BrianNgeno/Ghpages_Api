# GitHub-Search
This is an online application that outputs profile info of a github user and their reposirories on github on search.

## Specifications
When a user loads the pages, you can just search through to get the github users url and repos information
## Setup/Installation Requirements
* This page can be viewed  on any web browser available on your device.

## Known Bugs
There are no unresolved issues in regards to this code that I know of.

## Prerequisites
To be able to run this web application, you will need to have a web browser, preferably Google Chrome.
Just open the url link deployed on GitHub and run it.
